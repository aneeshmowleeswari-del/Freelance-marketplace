# Freelance-market# Freelance Project

## Overview
A web application to connect **clients** with **freelancers**.  
Users can register, post projects, submit proposals, and manage workflows efficiently.

## Features
- User registration & login (Client & Freelancer roles)
- Clients can post projects and view proposals
- Freelancers can submit proposals and manage profiles
- Admin can approve clients and freelancers
- Responsive design for desktop & mobile

## Technologies
- Backend: Django
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Version Control: Git, GitHub
